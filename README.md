### Bob's Screw Shop

In Bob's shop, beside other products, this one are currently in offer:
-screw: 5 cent piece
-screw nut: 3 cent piece
-washer: 1 cent piece

Write a program, which ask for the number of screws, screw nuts and washes,
calculate the total price.
The program should check the order: it is a mistake to order more screws
as screw nuts. In this case the program should notify the user: 
"Please, check your order!".
If the order is ok, the program should print:
"The order is ok!" and the total price.
